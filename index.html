Quais os comandos para configurar usuário e email no git?.

git config --global user.name "w3schools-test" git config --global user.email "test@w3schools.com"

Qual a diferença entre git e github?

git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo. GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões.

O que faz o comando commit?

O comando git commit captura um instantâneo das mudanças preparadas do projeto no momento.

O que faz o comando push?

O comando push transfere commits do repositório local a um repositório remoto.

O que faz o comando pull?

O comando git pull é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
