Quais os comandos para configurar usuário e email no git?.

git config --global user.name "w3schools-test" git config --global user.email "test@w3schools.com"

Qual a diferença entre git e github?

git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo. GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões.
